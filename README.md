# kasharaylo.com — v2.0

Personal site of Constantine Kasharaylo · QA Director.

A single-page static site built with vanilla HTML, CSS, and a touch of JavaScript. Test-runner-inspired aesthetic with Playwright green accents.

## Stack

- Pure HTML / CSS / vanilla JS — no build step, no dependencies
- JetBrains Mono + Inter (Google Fonts)
- Hosted via GitHub Pages

## Local preview

Just open `index.html` in a browser. There's no build.

```bash
# Or if you prefer a local server:
python3 -m http.server 8080
```

## Deploy

Pushed to `master` → automatically deployed via GitHub Pages.
